# MMOInventory Default Resource Pack
This is the public repository for the default MMOInventory resource pack.

## Download

In the top right corner of the repository, click the `Code` button, then under `Download source code`, select `ZIP`. You will download a ZIP version of the resource pack.

If you have Git installed, you can also navigate to your resource pack folder and locally clone the repository using `git clone git@gitlab.com:phoenix-dvpmt/mmoitems-default-resource-pack.git` or `git clone https://gitlab.com/phoenix-dvpmt/mmoinventory-default-resource-pack.git`

## Content

> Asset located at `assets/mmoinventory/textures/ui/oraxen.png` was largely inspired by a texture from the [Oraxen](https://www.spigotmc.org/resources/72448/) plugin. Credits to the original author. 

Textures and models for the inventory slots, compatible with the default MMOInventory configuration files. Please refer to [this wiki page](https://gitlab.com/phoenix-dvpmt/mmoinventory/-/wikis/Custom-Inventory-Texture) to learn how to apply custom textures to the inventory UIs.
